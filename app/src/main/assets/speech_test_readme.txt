语音识别测试功能使用说明
========================

功能概述
--------
本应用已集成whisper.cpp语音识别库，提供语音转文本功能。

主要功能
--------
1. 主界面新增"语音测试"按钮
2. 点击按钮跳转到语音测试界面
3. 语音测试界面包含：
   - 录音按钮（点击开始/停止录音）
   - 文本显示区域（显示识别结果）
   - 状态信息显示

使用步骤
--------
1. 在主界面点击"语音测试"按钮
2. 等待语音识别初始化完成
3. 点击录音按钮开始录音
4. 再次点击录音按钮停止录音
5. 等待语音转文本处理完成
6. 查看识别结果

技术实现
--------
- 语音识别库：whisper.cpp
- 模型文件：ggml-small.en-q5_1.bin
- 录音格式：WAV
- 依赖注入：Hilt
- UI框架：Jetpack Compose

文件结构
--------
com.syj.geotask.speech123/
├── SpeechToTextManager.kt    # 语音转文本管理器
├── SpeechTestViewModel.kt     # ViewModel
└── SpeechTestScreen.kt       # UI界面

权限要求
--------
- RECORD_AUDIO: 录音权限
- WRITE_EXTERNAL_STORAGE: 文件写入权限

注意事项
--------
1. 首次使用需要授予录音权限
2. 语音识别需要加载模型文件，初始化可能需要几秒钟
3. 录音时长建议控制在30秒以内
4. 识别效果取决于录音质量和环境噪音

故障排除
--------
如果遇到问题：
1. 检查录音权限是否已授予
2. 确认模型文件是否存在
3. 检查设备存储空间是否充足
4. 重启应用重新初始化
